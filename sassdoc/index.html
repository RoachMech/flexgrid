    General

    variables

    flexgrid-gutter-base

    $flexgrid-gutter-base: 2rem !default;

    Description

    Base gutter size for the grid. This size will be used if no other is specified when including the mixin.

    Author

    • RoachMech

    mixins

    flexgrid-container

    @mixin flexgrid-container() { ... }

    Description

    Make the container for the grid

    Parameters

    None.

    Example

    Basic Usage Sass

    .gallery {
    @include flexgrid();
}

    Basic Usage CSS Output

    .gallery {
    display: flex;
    flex-flow: row wrap;
    margin: 0 -1rem;
}

    Author

    • RoachMech

    flexgrid-item

    @mixin flexgrid-item() { ... }

    Description

    Make an item for the grid

    Parameters

    None.

    Example

    Basic Usage Sass

    .gallery__item {
    @include flexgrid-item(4);
}

    Basic Usage CSS Output

    .gallery__item {
    flex: 1 1 25%;
    max-width: 25%;
    padding: 0 1rem 2rem;
}
